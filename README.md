# 🚀 chronoNote – Take U Forward Calendar

A modern, interactive productivity calendar designed to help users **track daily progress, maintain streaks, and grow consistently** 🚀

Built with a focus on **self-improvement, consistency, and clean UI**, chronoNote is more than a calendar — it's your **daily growth companion**.

---

## ✨ Features

### 📅 Smart Interactive Calendar

* Monthly calendar view with smooth navigation
* Click on any date to add notes/tasks
* Clean and minimal UI for better focus

### 📝 Daily Notes System

* Add, edit, and delete notes for each day
* Data stored using **localStorage** (no backend required)
* Instant retrieval on revisit

### 🔥 Take U Forward Streak Tracker

* Mark a day as **"Productive"**
* Tracks continuous streaks automatically
* Displays:

  > 🔥 X Day Growth Streak
* Highlights streak days in **green shades**
* Motivates consistency and discipline

### 🌄 Dynamic Monthly Themes

* Each month displays a **unique background image**
* Themes include:

  * Nature 🌿
  * Seasons 🌸☀️🍁❄️
  * Motivation 💡
  * Productivity setups 💻
* Enhances visual experience and engagement

### 🎯 Productivity Insights

* Visual tracking of productive days
* Helps users build habits and stay accountable

### 💾 Local Storage Support

* No login required
* All notes and streak data saved in browser
* Fast and lightweight performance

---

## 🛠️ Tech Stack

* **Frontend:** React.js, Next.js
* **Styling:** Tailwind CSS
* **State Management:** React Hooks
* **Storage:** localStorage

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Afra245/calender.git
cd interactive-calendar
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Run the Project

```bash
npm run dev
```

### 4️⃣ Open in Browser

```
http://localhost:3000
```
