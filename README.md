# chronoNote-Calender-
=======
## Wall Calendar App

A modern, interactive wall-calendar style web application built with **Next.js (App Router)**, **React**, **TypeScript**, and **Tailwind CSS**.

### Features

- **Wall calendar UI** with a large monthly hero image, tactile calendar grid, and lined notes panel.
- **Dynamic monthly themes** driven by a local configuration mapping each month to its own image and colorway.
- **Date range selection** (start, end, and in‑between days highlighted) with sensible edge‑case handling.
- **Daily context panel** showing mock holidays, fun facts, and prompts for selected days.
- **Notes section** with lined‑paper styling, auto‑saved to `localStorage` with no backend.
- **Responsive layout**: side‑by‑side on desktop, vertically stacked on smaller viewports.

### Getting started

```bash
npm install
npm run dev
```

Then open `http://localhost:3000` in your browser.

### Assets

Place 12 nature images in `public/images` that match the filenames referenced in `src/lib/monthThemes.ts` (you can swap these paths for your own images as desired).
